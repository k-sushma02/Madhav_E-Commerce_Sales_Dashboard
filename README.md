# Madhav_E-Commerce_Sales_Dashboard
OVERVIEW:
1. Developed an interactive Power BI dashboard to track and analyze online sales performance across multiple dimensions.
2. Built a robust data model by creating relationships between Orders and Details tables using Order ID.
3. Created key KPIs including Total Sales, Profit, Quantity, and Average Order Value (AOV).
4. Implemented drill-down analysis using filters and slicers (Quarter, State) for dynamic data exploration.
5. Designed and customized multiple visualizations - Bar charts, Donut charts, Pie charts, Line and Area charts for trend analysis, Clustered charts for category         comparison
6. Performed data transformation and calculations to support user-driven insights and reporting.
7. Enabled interactive reporting with user-controlled parameters and slicers.
8. Resolved data challenges such as missing values and blank visuals due to relationship issues, incomplete time-based analysis (month-level gaps)
9. Improved dashboard usability with structured layout, color themes, and business-focused visuals.
